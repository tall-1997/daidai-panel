# v2.2.14

> 多设备会话管理、视图管理增强、定时停止多规则、多项修复。

## 新功能

### 多设备会话管理

- 新增「网页端会话上限」和「APP 端会话上限」配置（系统设置 → 账号安全 → 会话管理）
- 支持 1-20 个设备同时在线，超出上限时自动踢掉最早的会话
- 默认值为 1，与之前行为一致

### 视图管理增强

- 视图管理弹窗新增**编辑**和**删除**按钮，可直接在管理弹窗中操作视图
- 删除操作带确认弹窗，编辑操作打开编辑表单

### 定时停止多规则

- 定时停止输入升级为类似定时运行的多规则组件
- 支持添加/删除多条停止 cron 规则
- 每条规则实时验证并显示描述和下次停止时间

## 修复

- **config.sh 保存后重新打开为空** — 前端响应字段读取错误（`res.data?.content` → `res.content`）
- **定时停止 cron 编辑时消失** — 后端 `ToDict()` 方法遗漏 `stop_schedule` 字段
- **备份内容项 task_views 校验失败** — 定时备份白名单和默认值缺少 `task_views`
- **面板日志时间戳和级别重复显示** — GIN 日志格式化后经 `log.Print` 再次被标准库添加前缀，改为直接写入过滤器
- **备份操作按钮换行显示** — 操作列宽度不足，加宽至 260px 并禁止换行
- **表格复选框勾选图标未居中** — 18px 复选框的 `::after` 伪元素位置修正

- README 更新至 v2.2.14，补充定时停止和多设备会话管理功能描述
